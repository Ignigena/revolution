Using .Mac Backup

You can conveniently backup your entire ProWorship song library to your .Mac iDisk, keeping it safe in case something happens. Bundled with ProWorship is a QuickPick bundle that can be used with Apple's Backup software to automatically backup your library at the times you specify.

To backup your library to .Mac:

  

Install Apple's Backup program, available free to .Mac subscribers (Download Here.)

  

Launch Backup and create a new backup plan, or edit an existing one in the main window.

  

Click the Add (+) button below the Backup Items list.

  

Click the QuickPicks tab and check the box next to ProWorship.

Using Apple's Backup program and the free ProWorship plugin, you can backup your entire ProWorship library automatically and effortlessly, and never have to worry about losing your important data.
